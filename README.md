# Arevaham Pizza Bar

## How to run and see the site

**Option 1 – Python (if you have Python installed)**  
In a terminal, go to the project folder and run:

```bash
cd /Users/brsoyan/Desktop/arevaham
python3 -m http.server 8000
```

Then open in your browser: **http://localhost:8000**

**Option 2 – Node (if you have Node.js)**  
In the project folder:

```bash
npx serve .
```

Then open the URL shown in the terminal (e.g. http://localhost:3000).

**Option 3 – Open the file**  
Double‑click `index.html` to open it in the browser. The menu may not load from JSON when opened as a file; use Option 1 or 2 for the full menu and correct behavior.
